Sample : Coffee Shop
===================

Introduction
============

The coffee shop sample will explain RESTful application features with Jaggery. 
It is a nice client end user interface where user can make order for coffee. 
There are main seven REST calls happen in this application with PUT, POST, GET and DELETE. 
This sample contains database connection, URL mapping and web request in jaggery


Building the Service
====================

1. Use "ant" command in the CARBON_HOME/samples/jaggerySamples/ to copy jaggery apps.
2. This will copy the jaggery samples to the target directory and copy it to the 
<CARBON_HOME>/repository/deployment/server/jaggeryapps directory. If you start AppServer, Jaggery applications will be
available as a deployed service.
