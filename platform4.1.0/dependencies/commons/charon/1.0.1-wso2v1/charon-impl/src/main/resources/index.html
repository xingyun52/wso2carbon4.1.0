Welcome to WSO2 Charon

WSO2 Charon is the implementation of Simple Cloud Identity Management (SCIM) specification which is provided under Apache 2.0 License.

This will be integrated to WSO2 Identity Server to support SCIM based identity provisioning.

For more details about WSO2 Charon, please refer About Charon.

This webapp represents the reference implementation of SCIM service provider which is shipped with WSO2 Charon.

This is specifically hosted here to provide public SCIM endpoints for the 1st SCIM Interop Event which is taking place in parallel to IETF 83rd meeting.

Details for the interop testing with WSO2 Charon, refer Details for Interop.
