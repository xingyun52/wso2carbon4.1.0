Server Information (Mediation) Gadget

Introduction

This gadget shows real-time information such as number of requests, number of responses, number of faults, average/minimum/maximum response times of sequences, endpoints and proxy services of a given server. Inner circle represents the chords for sequences, endpoints and proxy services. Outer circle represents each sequence, endpoint and proxy service. Simply point the mouse over the graph to see extended information.

Configuration

Please click on the configuration tab and select the server to monitor from the drop down.
