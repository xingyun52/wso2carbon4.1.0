Server Information Gadget

Introduction

This gadget shows realtime information such as number of requests, number of responses, number of faults, average/minimum/maximum response times of services and operations of a given server. Each chord is proportional to the number of requests. Inner circle represents the services, while outer circle represents operations. Simply point the mouse over the graph to see extended information.

Configuration

Please click on the configuration tab and select the server to monitor from the drop down.
Note: This gadget is not applicable for mediation servers(WSO2 ESB).
