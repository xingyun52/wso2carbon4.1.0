Average Response Times of Services Gadget

Introduction

This gadget shows the average response time of each service. The graph updates for every 30 seconds and the time is presented over X-axis (in seconds) where the corespondent average response time is presented over Y-axis (in milliseconds). Further, it shows history data up to 270 seconds. Advanced tools available in the "canvas" (maximized) mode.

Configuration

Please click on the configuration tab and select the server to monitor from the drop down.
