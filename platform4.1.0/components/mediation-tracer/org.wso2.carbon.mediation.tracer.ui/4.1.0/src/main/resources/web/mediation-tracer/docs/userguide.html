Mediation Tracer

Traces the entire path when a message travels along a mediation sequence.To see the trace messages, you have to enable tracing for a sequence. Warning: Turning on the Tracer feature will have a drag on the performance. This is because all the messages will always be completely built (i.e., deferred building will not be done) and stored in the database by WSO2 Data Services. Hence this option should be used with caution.

Enabling Tracing

  1. In the navigator, under Manage/Mediation, click Sequences. The Mediation Sequences page appears.
  2. Click Enable Tracing.When tracing is enabled, the linkwill display Disable Tracing.

Figure 1: Enable tracing

Viewing Trace Messages

  1. In the navigator, under Monitor, click Message Tracer. The Mediation Message Tracer page appears. You can use the search function to find specific log messages for debugging purposes.
