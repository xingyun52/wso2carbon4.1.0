Priority Executors

Use the Priority Executors page to add or edit Priority Executors.

As shown in Figure,1 the sequence management section displays a list of existing Priority Executors.

Figure 1: Priority Executor Management

  • Edit
    • Click Edit to edit an existing executor.
  • Remove
    • Click Remove to remove an executor

Adding a Priority Executor

  1. On the Priotity Executors page, click Add. The Design Priority Executor page appears.
  2. Enter a Priority Executor name and other options and then save.

Editing a Priority Executor

  1. On the Priority Executors page, click Edit. The Priority Executors Design page appears, displaying the selected executor.

    2. Figure 3: Editing an Executor

Priority Executor Add/Edit options

When adding or editing a Priority Executor user can configure various options

Field Description
Name Name of the Executor
FixedQueus Weather fixed size queues are used or not
Queues Individual Queue Configurations
Max Maximum Number of Threads in th Executor
Core Core Number of Threads in th Executor
Keep-Alive Keep Alive time for Threads

Queue Configurations

Each and every priority has a queue associated with it. A message is put in to one of the queues corresponding to its priority.

Field Description
Size Size of the Queue, this option is visible only if fixed size queues are selected
Priority Priority of the Queue
