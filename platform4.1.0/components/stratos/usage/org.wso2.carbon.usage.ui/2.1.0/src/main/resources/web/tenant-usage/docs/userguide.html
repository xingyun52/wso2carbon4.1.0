View Usage

You can view usage of the stratos resource by your organization(Tenant) from this page

Figure1: View Usage

Description of usage data report fields

The Form Field Description
Report Duration The year and the month the usage is metered.
Number of Users Number of users of your stratos account.(This is valid only for current month)
Storage Usage The registry data storage amount.
Registry Bandwidth Usage Registry bandwidth consumed by the registry resource accesses(registry uploads, downloads).
Service Bandwidth Usage The bandwidth consumed by the web services hosted by you.
Webapp Bandwidth Usage The bandwidth consumed by the web services.
Service Usage Statistics Service statistics of requests and responses of services.

Table1: Description of usage data report fields
