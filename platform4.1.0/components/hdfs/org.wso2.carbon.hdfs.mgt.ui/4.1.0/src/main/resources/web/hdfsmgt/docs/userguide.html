HDFS - User Guide

WSO2 Carbon HDFS feature provides Apache HDFS based highly scalable second-generation distributed File System. HDFS supports unstructured data ( files ).

WSO2 Carbon HDFS helps user to create and manage a distributed file system with authentication and authorization. HDFS component uses Carbon User Manager as the user base.

HDFS feature comes with a management user interface. Management interface allows user to manage File System Objects.

Manage File System Objects

File System Objects management allows user to list available files and folders. User can create and delete File System Objects using interface.

Add File System Objects

    Home-> Manage-> HDFS File System Objects-> Add

List File System Objects

    Home-> Manage-> HDFS File System Objects-> List
