The Dashboard

The Dashboard is an Executive Information System, which gives users a Bird's Eye view of their SOA by way of a set of Gadgets. Users can use the default set of Gadgets shipped with a WSO2 product as well as adding their own.

Note: Users without proper Role permissions may not be able to manipulate Gadgets in System Dashboards. Currently, in order to manipulate a system Dashboard, users must belong to the Admin Role.

Gadgets

    WSO2 Gadgets adhere to the Google Gadget Specification. This allows users to extend the default Dashboard by adding new Gadgets.

  • Adding a Gadget

    1. Click "Add Gadgets"
    2. Enter the URL of the Gadget XML

  • Configuring Gadgets

    • By default the gadget authors can set gadget preferences for each gadget. These preferences can be viewed and modified by the users by opening the settings drop down in gadgets preferences.

  • Copy a gadget

    1. Click on settings button
    2. From the drop down menu, select "copy this gadget "

  • Move a gadget

    1. Click on settings button
    2. From the drop down menu, select "Move To "
    3. Choose the tab that the gadget should be moved

Tabs

    To minimize clutter, the Dashboard supports multiple Tabs.

  • create new Tab

    1. Click "Add a New Tab"
    2. Enter a name for the new Tab
    Once a Tab is created, users can select the newly created Tab and add Gadgets to it as described above.

  • Cloning a Tab

    • Dashboard users can clone an existing tab. To clone a tab;
    1. Click "Clone Active Tab"
    2. Enter a name for the new Tab
    Once the Clone is created users can configure the gadgets.

Finding Gadget Details

Once the gadget is added, the user can inquire the gadget details by clicking on about gadget in the preferences menu.
  1. Click on settings button
  2. From the drop down menu, select "About Gadget "
