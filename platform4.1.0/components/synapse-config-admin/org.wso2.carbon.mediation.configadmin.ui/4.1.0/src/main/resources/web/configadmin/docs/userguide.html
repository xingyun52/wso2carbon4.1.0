Synapse Configuration

Use this feature to view your current system configuration, modify the configuration, and save the configuration to the system.

Changing the Configuration

  1. In the navigator, under Configure, click Synapse. The Synapse Bus Configuration page appears. It display the Synapse.xml file. Figure 1: Synapse Configuration
  2. Modify the configuration as required, and click Update.
  3. If you want to undo the changes, clickReset.
