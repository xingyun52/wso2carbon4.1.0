Then Mediator

The 'Then' mediator always a child of a Filter mediator and it act as a container (or parent) to a sequence of other mediators.

This mediator act on a message if the regular expression or the xpath expression given in the parent filter mediator is matched with the message.

There are no mediator specific configurations in the 'Then mediator'.

Please add child mediators to this mediator and configure those mediators.
