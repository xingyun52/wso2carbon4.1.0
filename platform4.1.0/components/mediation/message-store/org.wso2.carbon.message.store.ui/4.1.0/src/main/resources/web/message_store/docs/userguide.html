Manage Message Stores

The Message Stores acts as storage for ESB messages.Store mediator can be used To Store messages That are going through the ESB Mediation Path.
  1. In the navigator, under Manage/Service Bus, click Message Stores.The Manage Message Stores page appears.

    2. Figure 1: Message Store option types

Adding Message Stores

Adding an In-Memory Message Store

  1. Click Add In Memory Message Store link. The Add In Memory Message Store page appears.

    2. Figure 2: Adding In-Memory Message Store

  2. Enter the required details, and click Save. The Manage Message Stores page will display the added Message Store.

    3. Figure 3: Message Store List

Adding a JMS Message Store

  1. Click Add JMS Message Store link. The Add JMS Message Store page appears.

    2. Figure 4: Adding JMS Message Store

  2. Enter the required details, and click Save. The Manage Message Stores page will display the added Message Store.

    3. Figure 5: Message Store List

Adding a Custom Message Store

  1. Click Add Custom Message Store link. The Add Custom Message Store page appears. You can add parameters to the message store by giving a Name and Value to Message Store Parameters and clicking Add Parameter button.

    2. Figure 6: Adding Custom Message Store

  2. Enter the required details, and click Save. The Manage Message Stores page will display the added Message Store.

    3. Figure 7: Message Store List

Message Store Operations

  1. When you click on an added Message Store list item, it will direct you to a page where Listed Messages appears. You can delete a message or delete all messages in a store.

    2. Figure 8: Messages List

  2. When you click Show Envelope of a message, the message content will be displayed.

    3. Figure 9: View Message Content
