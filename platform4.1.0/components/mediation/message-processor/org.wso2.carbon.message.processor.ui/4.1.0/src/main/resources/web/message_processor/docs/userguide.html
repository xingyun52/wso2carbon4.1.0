Manage Message Processors

The Message Processors are set of processors that are intended to consume the ESB messages stored in ESB Message stores and do processing.

  1. In the navigator, under Manage/Service Bus, click Message Processors.The Manage Message Processors page appears.

    2. Figure 1: Message Processor option types

Adding Message Processor

Adding a Scheduled Message Forwarding Processor

  1. Click Add Scheduled Message Forwarding Processor, Add Scheduled Message Forwarding Processor page appears.

    2. Figure 2: Adding Scheduled Message Forwarding Processor

  2. Enter the required details, and click Save. The Manage Message Processors page will display the added Message Processors.

    3. Figure 3: Message Processor List

Adding a Message Sampling Processor

  1. Click Add Message Sampling Processor, Add Message Sampling Processor page appears.

    2. Figure 4: Adding Message Sampling Processor

  2. Enter the required details, and click Save. The Manage Message Processors page will display the added Message Processors.

    3. Figure 5: Message Processor List

Adding a Custom Message Processor

  1. Click Add Custom Message Processor, Add Custom Message Processor page appears.

    2. Figure 6: Adding Custom Message Processor

  2. Enter the required details, and click Save. The Manage Message Processors page will display the added Message Processors.

    3. Figure 7: Message Processor List

Message Processor Operations

  1. Users can deactivate the defined message processors listed in the Manage Message Processors page.

    2. Figure 8: Deactivated Message Forwarding Processor

  2. Scheduled Message Processor Comes with a User interface to give user some manual control over message forwarding operations. To use the user interface It must be deactivated if its in the active state. Using the UI, User can delete, resend, delete all and resend all message in a message processor.

    3. Figure 9: Scheduled Message Forwarding Processor Operations
