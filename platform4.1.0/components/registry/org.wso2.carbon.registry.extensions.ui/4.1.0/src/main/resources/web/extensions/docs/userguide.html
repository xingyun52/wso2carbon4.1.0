Extensions

Add extension interface can be used to add extensions to the Governance Registry. Simply upload the JAR file containing the extension using the interface shown in Figure 1.

There are 3 types of extensions that you can add to the Registry using this interface.

  • Handlers
  • Filters
  • Aspects

Details on these extensions can be found in Governance Registry Developer's Guide

List Extensions

List extensions interface shown in Figure 1 can be used to list extensions added to the Governance Registry. It also provides a way to remove added extensions.
